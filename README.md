# Hyper-Spacechalk

> A theme for the [Hyper Terminal](https://hyper.is) inspired by Jessebot's  spacechalk VIM theme for [onboardme](https://github.com/jessebot/onboardme) 
## Install
Add `hyper-spacechalk` to the plugins list

```js
module.exports = {
  ...
  plugins: [
    'hyper-spacechalk'
  ],
}
```
